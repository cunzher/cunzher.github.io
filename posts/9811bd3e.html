PicPlus——极简好用的移动端图床工具

发表 2020-07-03 分类 随笔 热度： Valine：
本文字数： 899 阅读时长 ≈ 1 分钟

经常使用 MarkDown 进行写作的同学们一定对 Typora 这款软件不陌生，它是桌面端 Markdown 写作的不二之选，配合 PicGo 进行图文创作简直不要太方便。

不过，如果我想在外出期间进行灵感写作，总是带着笔记本似乎并不舒适，于是我有了《纯纯写作》，一款支持 Markdown 语法的声称永不丢失数据的移动端极简纯文本写作软件。不过，用它来写作，总是遇到一个问题：插图好麻烦。于是 … …

前段时间我在酷安看到了一款移动端的图床工具PicPlus，于是下载来把玩了一番。

这是 PicPlus 的主界面，极简风的设计直戳小心脏，几乎是瞬间就爱上了；

首页可以查看你之前上传过的图片，上传方式可以选择本地相册上传，当然也可以选择直接拍照上传以及链接上传；

展开侧栏，我们可以看到 PicPlus 的功能菜单，你可以在这里选择图床列表、进行数据的备份，浏览收藏的图片以及查看帮助文档；

和 PicGo 相比，PicPlus 可供选择的图床要多一些

数据备份主要是针对上述图床的配置进行备份，支持本地备份以及WebDAV备份。

我个人倾向于使用坚果云的WebDAV服务，简单高效。

此处的备份策略是付费功能，付费计划我们稍后再讲~

收藏夹里列出的是你收藏的图片，可以直接在这里浏览、拷贝、分享等等

值得一提的是，PicPlus 可以让你编辑 Markdown 图片标题，不是像其他软件一样直接使用文件名称，在图片详情中你可以查看关于这张图片的详细信息；

在设置页面，你可以进行较为全面的DIY，图片上传前确认、图片压缩开关、设置压缩质量以及格式化、自动复制；

你可以格式化为Markdown、Html、BBCode、Url

作者还提供了一些实验功能供体验，包括搜狗图床、图壳图床、自动打开相册等

对于免费用户，每个图床每月上传限制为20张图片，超过后需要等到下个月才能继续使用，付费后无限制；

激活码分为年付和永久买断两类，目前年费10元软妹币，永久28元；

对于一（tuo）般（geng）作者来讲，免费是足够使用的，不过，开发者这么酷，为什么不请他嚯一杯咖啡呢
